# Ходжаев Даян  тестовое задание To-do

## Описание
Это простое To-Do приложение, разработанное с использованием PHP Laravel для бэкенда и Bootstrap для фронтенда. Пользователи могут добавлять, редактировать, удалять и отмечать задачи как выполненные. Приложение использует SQLite для хранения данных.

## Установка и запуск

### Предварительные требования
- Установленный PHP (версии 8.3 или выше)
- Установленный Composer

### Шаги по установке

1.Переместите папку проекта к себе
2.Запустите миграции php artisan migrate
3.Сайт готов к эксплуатации !

### Функцианальность сайта

С самого начало вас встречает страница входа ,нужно зарегистрироваться .

Аутентификация полностью реализована с помощью пакета Laravel breeze

В вертске использовался Bootstrap 5, всего есть несколько страниц а именно
    вход

    регистрация

    Главная

    Добавление задачи

    Редактирование задачи

Сверху в навигационной панели представлена кнопка для перехода на главную, кнопка для выхода из аккаунта
и строка для поиска по id или же названию задачи

Над задачами можно совершать все CRUD операции а именно
1.Создание
2.Редактирование
3.Удаление
4.Общий список

Только ВЫ видите свои задачи и больше никто
